## Add xtask code graph generator

Commit date: 2026-06-07
Source commit: this commit

Added a local `xtask` workflow for generating Rust code navigation artifacts
from the current workspace.

Implementation:

- added `cargo run -p xtask -- code-graphs [--out-dir <dir>]`;
- generated Mermaid and DOT workspace crate graphs from `cargo metadata`;
- generated Mermaid and DOT internal crate dependency graphs;
- rendered SVG visualizations from the DOT graphs via Graphviz `dot`;
- added curated IR overview graphs for `boxes`, `signals`, `fir`, and `ui`;
- added a lightweight public API source-scan index for public structs, enums,
  traits, functions, type aliases, constants, modules, and re-exports;
- documented the workflow in `crates/xtask/README.md` and linked the generated
  artifacts from the top-level README.

Generated artifacts:

- `docs/code-graphs/workspace-crates.mmd`
- `docs/code-graphs/workspace-crates.dot`
- `docs/code-graphs/workspace-crates.svg`
- `docs/code-graphs/internal-crate-deps.mmd`
- `docs/code-graphs/internal-crate-deps.dot`
- `docs/code-graphs/internal-crate-deps.svg`
- `docs/code-graphs/ir-overview.mmd`
- `docs/code-graphs/ir-overview.dot`
- `docs/code-graphs/ir-overview.svg`
- `docs/code-graphs/public-api-index.md`

Validation:

- `cargo fmt --all`
- `cargo check -p xtask`
- `cargo clippy -p xtask --all-targets -- -D warnings`
- `cargo run -p xtask -- code-graphs`
- `cargo run -p xtask -- code-graphs --out-dir /private/tmp/faust-rs-code-graphs-check`
