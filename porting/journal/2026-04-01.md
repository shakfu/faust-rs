## fix(tests): drop accidental `stdfaust.lib` imports from self-contained corpus repros

Commit date: 2026-04-01
Source commit: working tree after `362b258`

Removed unnecessary `import("stdfaust.lib");` lines from the self-contained
corpus reproducers that only exercise core language/runtime primitives. These
fixtures were compile-path regressions, not library-loading tests, so the
imports created avoidable CI failures on runners without a configured Faust
library search path.
