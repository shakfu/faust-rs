## docs(porting): define RAD E1 block-local boundary semantics

Commit date: 2026-04-28

Source commit: this commit

Documented the first intended user-visible semantics for RAD phase E1 over
accepted LTI recursive groups:

- `rad(expr, seeds)` uses the current Faust compute block length as the finite
  reverse window (`count` / `ma.BS`).
- The terminal adjoint state at the end of the block is zero.
- No adjoint state persists across block boundaries in the first E1
  activation.
- The resulting gradient is exact for a block-local objective and explicitly
  truncated for longer stream objectives.

Validation:

- `cargo fmt --all`
- `cargo clippy --workspace --all-targets -- -D warnings`
- `cargo test --workspace --all-targets`

## feat(propagate): RAD phase E0 recursive-linearity classifier

Commit date: 2026-04-28

Source commit: this commit

Implemented the read-only feasibility classifier from
`porting/reverse-ad-rad-implementation-plan-2026-04-27-en.md` §19.6,
Phase E0.

Key changes:

- Added `crates/propagate/src/stateful_rad.rs` with public Rustdoc for
  `RadRecLinearity` and `classify_de_bruijn_rec_group(...)`.
- The classifier walks `DEBRUIJNREC` bodies structurally and classifies
  dependence on the current recursive back-edge as:
  - `LinearLti` for affine recursive dependence under literal coefficients,
  - `LinearTimeVarying` for affine recursive dependence under signal-dependent
    coefficients,
  - `Nonlinear` for transcendental, branch, discrete, or multiplicative
    dependence on recursive state.
- Added focused unit tests for constant-coefficient linear recursion,
  signal-dependent coefficients, nonlinear recursive use, and `delay1` as an
  LTI shift, plus a multi-output cross-coupled LTI recursive group.
- Updated RAD documentation and the supported-subset snapshot to state that E0
  is implemented but does not make recursive `rad(...)` legal yet.
- Cleaned up existing RAD/FAD runtime-test clippy warnings so the workspace
  `-D warnings` gate remains usable.

Validation:

- `cargo fmt --all`
- `cargo test -p propagate stateful_rad`
- `cargo clippy --workspace --all-targets -- -D warnings`
- `cargo test --workspace --all-targets`
