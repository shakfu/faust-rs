## feat(rad): build grouped recursive adjoint frontier

Commit date: 2026-05-06
Source commit: `445e63e`

Added the next phase-E1 bridge expected by the public RAD sweep: a frontier
helper that receives recursive projection cotangents, groups them by source
recursion, and returns ordered adjoint projections.

### Changes

- Added `build_lti_recursive_adjoint_projections`.
- The helper ignores non-eligible entries, preserving existing diagnostics for
  unsupported recursive families.
- Eligible projections that read the same LTI `DEBRUIJNREC` now share one
  `ReverseTimeRec` group.
- Added a regression proving frontier order is preserved and the shared
  reverse-time group carries the expected cotangents.

### Remaining E1 work

- Collect the projection-cotangent frontier from the actual reverse sweep.
- Merge returned reverse-time adjoint projections into per-sample gradients for
  active LTI coefficients/seeds.

### Validation

- `cargo test -p propagate lti_recursive_projection_frontier_shares_reverse_group`
- `cargo test -p propagate lti_recursive_adjoint_group_combines_slot_cotangents`
- `cargo fmt --all`
- `cargo clippy -p propagate --all-targets -- -D warnings`

## feat(rad): group LTI recursive adjoint cotangents

Commit date: 2026-05-06
Source commit: `f662f2a`

Continued phase E1 by adding the shared recursive-adjoint grouping primitive
needed before public `rad(...)` can lower recursive projections.

### Changes

- Added `build_lti_recursive_adjoint_group`, which accepts multiple
  `(slot, cotangent)` entries for one LTI `DEBRUIJNREC`.
- Duplicate slot cotangents are accumulated with signal addition.
- The single-projection bridge now delegates to the grouped helper.
- Added a regression for a two-lane recursive group with duplicate cotangents
  on one lane.

### Remaining E1 work

- Make the public RAD sweep discover all projection cotangents for a recursive
  group and call the grouped helper.
- Route reverse-time state adjoints into per-sample gradients for active LTI
  coefficients/seeds.

### Validation

- `cargo test -p propagate lti_recursive_adjoint_group_combines_slot_cotangents`
- `cargo test -p propagate lti_recursive_projection_builds_reverse_time_adjoint_projection`
- `cargo fmt --all`
- `cargo clippy -p propagate --all-targets -- -D warnings`
