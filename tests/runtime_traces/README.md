# Runtime Trace Snapshots

Golden runtime traces generated by the `xtask` `interp` trace harness.

Current status:
- Phase 2 scaffold is implemented (`interp-trace-gen` / `interp-trace-check`)
- snapshots are stored under `rust/<case>/<scenario>.json`
- comparison is currently exact JSON (deterministic renderer), with tolerance-
  based numeric comparison planned in a later Phase 2 increment

Source fixtures:
- `tests/runtime_corpus/`

Commands:

```bash
cargo run -p xtask -- interp-trace-gen
cargo run -p xtask -- interp-trace-check
```
